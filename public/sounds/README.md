# Sounds Directory

Place your drum kit sample files here.

## Required Files

- `kick.wav` - Kick drum sample
- `snare.wav` - Snare drum sample  
- `hihat.wav` - Hi-hat sample
- `cowbell.wav` - Cowbell sample

## File Format

- Format: WAV
- Sample Rate: 44.1kHz recommended
- Channels: Mono or Stereo
- Bit Depth: 16-bit or 24-bit

If these files are not present, the application will use generated placeholder sounds.
